As you can see there are three JS files in MaintenaceRequest directory
1. maintenance-request-add.js - This file will contain all that code which are specifically related to the Add functionality of Maintenance Request
2. maintenance-request-update.js - This file will contain all that code which are specifically related to the Update functionality of Maintenance Request
3. maintenance-request.js - This file will contain all that code which are common for Add, Update functionality example is Save functionality which will 
   remain the same for both of them and many more
